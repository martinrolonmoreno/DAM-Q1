Listas de términos

Glosario

HTTP
Protocolo de Transferencia de Hipertexto, un protocolo que establece las reglas para la comunicación entre un navegador (cliente) y un servidor web. Es la base de la World Wide Web y funciona como un modelo de "solicitud-respuesta", donde el navegador solicita información (como una página web) y el servidor responde con los datos correspondientes.
HTML
Lenguaje de Marcado de Hipertexto, lenguaje estándar utilizado para estructurar y organizar el contenido de las páginas web. Su función principal es definir la estructura de un documento web utilizando un sistema de etiquetas para organizar elementos como párrafos, encabezados, imágenes y enlaces, haciendo posible que el navegador muestre el contenido.
URL
Una URL (Localizador Uniforme de Recursos) es la dirección única y específica de un recurso en internet, como una página web, una imagen o un archivo, que se utiliza para acceder a él.
TCP/IP
TCP/IP (Protocolo de control de transmisión/Protocolo de Internet) es un conjunto de protocolos de comunicación que permite a los ordenadores y dispositivos conectarse e intercambiar datos de manera fiable en redes como Internet.
Internet
Red informática mundial, descentralizada, formada por la conexión directa entre computadoras mediante un protocolo especial de comunicación.
W3C
El W3C (World Wide Web Consortium) es un consorcio internacional que desarrolla estándares y directrices para la Web con el objetivo de garantizar su crecimiento a largo plazo, asegurando que sea universal, accesible y funcione en cualquier dispositivo.
